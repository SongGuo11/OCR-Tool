OCR文字识别工具使用说明

首次使用步骤：
1. 确保电脑已连接网络
2. 双击运行 install.bat 安装必要的环境和依赖
3. 安装完成后双击 run.bat 运行程序

获取百度OCR配置：
1. 访问 https://console.bce.baidu.com/ai/
2. 注册/登录百度账号
3. 创建应用获取 APP_ID、API_KEY 和 SECRET_KEY

使用说明：
1. 输入百度OCR的配置信息
2. 点击"启动服务"按钮
3. 按Ctrl+Alt+D进行截图识别
4. 按右键取消截图

注意事项：
- 首次运行时间可能较长，请耐心等待
- 如果运行出错，请尝试重新运行install.bat
- 需要Python 3.11或更高版本
